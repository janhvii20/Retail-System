# Retail Sales Frontend

React-based UI for the Retail Sales Management System.

## Setup

```bash
npm install
```

## Running

```bash
npm run dev    # Development server on http://localhost:3000
npm run build  # Production build
npm run preview # Preview production build
```

## Features

- Full-text search across customer names and phone numbers
- Multi-select filtering by region, gender, age, category, tags, payment method, and date
- Sorting by date, quantity, and customer name
- Pagination with 10 items per page
- Responsive design matching Figma specifications
